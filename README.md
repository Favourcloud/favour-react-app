# Favour React App

A React application deployed on an Ubuntu EC2 instance using Nginx as part of the DevOps Micro Internship Assignment 2.

## Technologies Used

- React
- Vite
- JavaScript
- Node.js
- npm
- Ubuntu Server
- AWS EC2
- Nginx
- Git & GitHub

## Deployment

The application was:

1. Developed using React and Vite
2. Built using npm
3. Deployed on an Ubuntu EC2 instance
4. Served using Nginx
